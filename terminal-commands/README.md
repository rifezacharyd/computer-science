# Top 7 Terminal Commands Every Linux User Should Know

A practical reference project covering essential terminal commands organized by category, with guides and interactive exercises.

## Command Categories

| Guide | Commands Covered |
|-------|-----------------|
| `guides/navigation.md` | cd, ls, pwd, find |
| `guides/file_operations.md` | cp, mv, rm, mkdir, touch |
| `guides/text_processing.md` | grep, sed, awk, cat, less |
| `guides/system_info.md` | top, htop, df, du, free, uname |
| `guides/permissions.md` | chmod, chown, umask |
| `guides/networking.md` | curl, wget, ping, ss, ip |
| `guides/process_management.md` | ps, kill, jobs, bg, fg, nohup |

## Practice

chmod +x practice/exercises.sh
